第一节：搭建环境

环境准备

  • win7 iso
  • win2003 iso
  • kali linux
  • wooyun

软件包准备

  • phpstudy
  • java jdk
  • python 2.x/3.x
  • waterfox/firefox

搭建环境

  1. win7

  • 从网上找到适合win7的密匙

  • 傻瓜式的默认就好了
  • 将软件包拷贝到win7虚拟机中
  • 如图就是火狐游览器
  • 在安装PHP study的时候显示缺少一个vc9，那么我们就找一个给他装上
  1. 安装wooyun
  • 将下载下来的压缩文件解压，并直接从虚拟机导入
  • 导入重启后获取他的ip地址
  • 耐心等待。。。。
  • 根据步骤去重新设置一下
  • 完美打开

小结

  • 在任何的学习开始之前都是枯燥的。忍耐，坚持之后才能看到通往彼岸的船。
